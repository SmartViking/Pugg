Pugg by SmartViking

A program that helps to learn new languages

This program is meant to help you memorize new words in new languages. If you've got homework where you have to learn new words in another language, you can create your own .pugg file and use it with the program, instructions for how to do that are located in the README file.

Video Demonstration

This video is from Pugg v0.1, which is not the latest version. The newest version is much better.
